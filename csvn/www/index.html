    • Gestion des dates dans Mantis
    Les dates sont stockées en format standard Unix ( nb de secondes depuis 01/01/1970 minuit UTC)
    pour les convertir en format Excel (nb de jours depuis 01/01/1900 minuit à l'heure UTC), il faut copier la formule
    = ( A1+(DATEVAL("1/1/1970") * (24*60*60) ))/ (24*60*60)
    Pour passer en GMT+2: (fuseau Paris)
    = ( A1+ 2209168800 )/ 86400
    et pour l'importation dans Mantis (Excel vers format Unix)
    =A1*86400-2209168800

    intégration Subversion dans Mantis (lien des modifs code source dans mantis)

    pour mettre un suivi du code source mettre le mot clé :bug, par exemple bug #1 exemple de lien sur la demande n°1
